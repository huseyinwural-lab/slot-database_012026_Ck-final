# Dokümantasyon (TR) — Tek Doğru Kaynak

Bu klasör, projenin **yetkili (authoritative)** dokümantasyon setidir.

## Yapı
- `runbooks/` — CI, release, operasyon runbook’ları
- `guides/` — kurulum, geliştirme, deploy rehberleri
- `admin/` — admin paneli, roller, yetkiler
- `api/` — API özetleri ve referanslar
- `architecture/` — üst seviye mimari
- `adr/` — Mimari Karar Kayıtları (ADR)

## Hızlı Linkler
- Operasyon dokümanları (legacy, zamanla buraya taşınacak): `/docs/ops/`
- Ödeme dokümanları (legacy, zamanla buraya taşınacak): `/docs/payments/`
- Kılavuzlar (legacy): `/docs/manuals/`

> Not: Çalışan sistemi bozmamak için mevcut `/docs/` içeriği olduğu gibi bırakılmıştır.
> Yeni dokümanlar `/docs/new/en` altında yazılmalı ve `/docs/new/tr` altında karşılığı tutulmalıdır.

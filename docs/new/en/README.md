# Documentation (EN) — Single Source of Truth

This folder is the **authoritative** documentation set for the project.

## Structure
- `runbooks/` — CI, release, operations runbooks
- `guides/` — install, development, deployment guides
- `admin/` — admin panel, roles, permissions
- `api/` — API overview and reference pointers
- `architecture/` — high-level architecture
- `adr/` — Architecture Decision Records
